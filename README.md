## 🧭 如何使用这个知识库

### 作为仓库所有者
1. **添加新内容**：
   - 在相应分类文件夹中创建新的Markdown文件
   - 使用下面的[笔记模板](#笔记模板)保持一致性
   - 提交时使用描述性信息（例如："添加React hooks学习笔记"）
   
2. **更新现有内容**：
   - 每季度回顾笔记，深化理解
   - 添加相关概念之间的新连接
   - 移除过时信息或标记为历史背景

3. **查找信息**：
   - 使用GitHub搜索（左上角）查找特定主题
   - 通过目录结构按类别浏览

## 💭 我的学习理念

### 核心原则
- **质量重于数量**：深刻理解少数概念，胜过肤浅了解许多概念
- **实践应用**：始终将理论与现实世界应用场景连接
- **间隔重复**：在渐增的时间间隔中回顾关键概念，将短期记忆转化为长期记忆
- **教学相长**：向他人（或未来的自己）解释概念能揭示理解上的漏洞
- **拥抱困惑**：陷入困境是学习过程的一部分 — 这意味着正在成长

### 当前学习目标（2025-2026）
1. **技术深度**：掌握使用现代框架的全栈开发
2. **设计思维**：培养强大的以用户为中心的设计能力
3. **沟通能力**：提高技术写作和演讲表达能力
4. **系统思维**：理解不同技术和概念如何相互连接
